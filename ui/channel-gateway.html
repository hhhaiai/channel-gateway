Channel Gateway

一个互通房间可放入 N 个 QQ / Telegram 群；任一群发言会转发给其余群。

连接

官方接入

Telegram

  1. 在 BotFather 创建 bot，取得 token。
  2. 按 OpenClaw 官方 Telegram 文档配置 token。
  3. 若群内无需 @bot：BotFather → /setprivacy → Disable。
openclaw channels status --probe

QQ 群

  1. 按 OpenClaw QQBot 官方文档创建机器人并取得 AppID:AppSecret。
  2. 安装/启用 QQBot channel plugin 后执行：
openclaw channels add --channel qqbot --token "AppID:AppSecret"

群 ID 填入 conversationId；to 是发送目标。不要把群 ID 写进 groupAllowFrom。

Channel 状态

连接后加载。

互通房间

每个房间至少两个 endpoint。例：1 QQ + N TG、N QQ + 1 TG 或 N QQ + N TG。保存后必须重启服务生效。
