# 资源感知并发配置实现

> 状态：实施文档
>
> 范围：只定义 Delivery Worker 全局并发默认值、配置优先级和硬上限；不包含网页编辑、Provider 限流或消息聚合。

## 1. 目标

同一套 JavaScript 服务需要同时支持：

- Docker 容器部署。
- 物理机或普通电脑直接部署。
- 低配置机器上的保守运行。
- 高配置机器上的有限扩展。
- 运营人员显式覆盖默认值。

系统不能使用无限并发，也不能长期写死同一个默认值。默认值必须根据当前进程实际可见的 CPU 和内存推导，并由统一硬上限约束。

## 2. 配置优先级

从高到低：

```text
OpenClaw plugin config
  deliveryMaxConcurrency
        ↓
环境变量
  CHANNEL_GATEWAY_DELIVERY_MAX_CONCURRENCY
        ↓
运行时资源探测默认值
```

显式 plugin config 适合未来网页控制台保存后的持久化配置。环境变量适合 Docker、systemd 和临时部署覆盖。两者都不存在时才自动探测。

## 3. 资源探测

CPU：

```js
os.availableParallelism()
```

该值由 Node/libuv 提供。在支持的容器环境中，它反映进程可用 CPU 或 affinity，而不是简单读取宿主机总核心数。

内存：

```js
process.constrainedMemory()
```

如果返回正数，使用容器或进程约束内存；如果返回 `0`，说明没有可识别约束，回退到：

```js
os.totalmem()
```

探测值必须是正安全整数。异常值不会被静默接受。

## 4. 默认值公式

```text
cpuBound = availableParallelism × 2
memoryBound = floor(memoryLimitBytes / 256 MiB)
autoDefault = clamp(min(cpuBound, memoryBound, 32), 1, 256)
```

说明：

- 每 CPU 两个发送 lane 是保守的网络 I/O 初始值，不是 CPU 线程数量。
- `256 MiB/lane` 是资源预算保护系数，不表示每个请求实际占用 256 MiB。
- 自动值最多为 8。早期上限 32 在 20 平台、200 endpoint 的 SQLite 压测中出现明显反向性能，自动值因此收紧；显式配置和环境变量仍可提高到 256。
- 运营人员可以显式提高，但所有入口的绝对硬上限都是 256。

示例：

| 可用资源 | 自动并发 |
|---|---:|
| 1 CPU / 256 MiB | 1 |
| 1 CPU / 512 MiB | 2 |
| 2 CPU / 4 GiB | 4 |
| 8 CPU / 8 GiB | 8 |
| 32 CPU / 128 GiB | 8 |

## 5. 环境变量

```bash
CHANNEL_GATEWAY_DELIVERY_MAX_CONCURRENCY=12
```

只接受十进制正整数字符串：

```text
1..256
```

拒绝：

- `0`
- 负数
- 小数
- `+4`
- `0x10`
- `12jobs`
- 大于 256 的数字

空白或未设置表示不覆盖自动值。

## 6. Plugin 配置

```json
{
  "plugins": {
    "entries": {
      "channel-gateway": {
        "config": {
          "deliveryMaxConcurrency": 12
        }
      }
    }
  }
}
```

Plugin 配置优先于环境变量，适合未来网页保存。Schema 保留 `minimum: 1` 和 `maximum: 256`，但不声明固定 default，避免 OpenClaw 在运行前把自动探测覆盖为常数。

## 7. 防御边界

同一个值经过两层校验：

1. 资源解析模块校验配置和环境变量。
2. `DeliveryWorker` 构造器再次强制 `1..256`。

即使插件 Schema 被绕过，也不能创建无限或超上限并发池。

## 8. 与后续功能的边界

本块不实现：

- 网页控制台编辑并发配置。
- 单平台或单账号限流。
- Token bucket、429 cooldown 和熔断。
- 高频消息聚合。
- AI 摘要。

后续网页控制台必须复用相同的 `1..256` 约束，并展示当前值来源：配置、环境变量或自动探测。
